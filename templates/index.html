DXC Client — QA Automation Prototype

AI in QA Automation

Proof of concept · built by Huseynbala Gurbanli

Human-in-the-loop — AI proposes, engineers approve. Nothing merges automatically.

Paste a failing test error/log. AI diagnoses root cause and proposes a fix.


Paste a PR/commit description. AI identifies which test scenarios are impacted and what updates they need.


Paste a captured API request/response (Fiddler-style). AI generates a test class.


Paste a test failure result. AI drafts a ready-to-file bug ticket.
